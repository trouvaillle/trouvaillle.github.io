# 관리자 계정 생성

```js
db.createUser({
  user: "관리자 아이디",
  pwd: "관리자 비밀번호",
  roles: [
    {
      role: "root",
      db: "admin",
    },
    {
      role: "userAdminAnyDatabase",
      db: "admin",
    },
  ],
});
```

# 컬렉션 CRUD

| CRUD   | 설명                                        |
| ------ | ------------------------------------------- |
| Create | db.createCollection("`컬렉션`")             |
| Read   | show collections                            |
| Update | db.`컬렉션`.renameCollection("`새 컬렉션`") |
| Delete | db.`컬렉션`.drop()                          |
