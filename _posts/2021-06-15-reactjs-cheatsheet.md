### React.js 프로젝트 만들기(Typescript)

```console
npx create-react-app 이름 --typescript
```

### Typescript 초기화(`tsconfig.json` 생성)

```console
npx typescript --init
```

### Typescript 모듈 설치

```console
yarn add @types/react @types/react-dom
yarn add -D typescript
```

### eslint, prettier 모듈 설치

```console
yarn add -D eslint prettier
```
